== Description ==

This plugin exports the feed stats of each wordpress blog in the Google Reader. the stats consists of total like count of a blog and number of it's followers with use of likekhore.com API. 

== Installation ==

1. Upload `wp-likekhor` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
